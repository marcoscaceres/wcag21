conventional user interface components

Proposed

widely recognized user interface component types

The Working Group seeks feedback on the lists below. These lists are likely to change as items are likely to be refined, removed or added.

Conventional names / conventional user interface component for buttons or controls:

compose
compose new item such as an email.
next
move to the next item in the series
delete
delete current item, selected control or text
previous
move to the previous item in the series
toc
opens a table of content
submit
submits the form data or request to the server
undo
revert to the state before the user's most recent changes
cancel
closes the dialog and discards any changes the user may have made within that dialog
reset
resets all the controls to their initial values
label
add label to the selected item or the item at the current location.
buy
buy the selected item or buy the item at the current location . Often involves adding the current item into a shopping cart.
add
add the selected item or the item at the current location to a list
move
move the selected item
save
save the current content
send
send a form, email or request
close
close current dialog
more
show more information
print
print current page, selection or the item at the current location
received
open received folder
sent
open sent folder
edit
edit current item, selection or the item at the current location
forward
forward to current item such as an email
reply
reply to current item such as an email
upload
upload file to the server
open
open new item
new
open new item
settings
open settings and options
mode
change the mode or functionality
higher
increases the level of the relevant setting
lower

help

copy

color

date

remove

right

left
moves a selected item to the left to another column or list
